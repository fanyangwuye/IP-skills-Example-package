# 最小文案到图片闭环说明

## 目标

先打通一个最小闭环：

1. `ip-copy-skill` 产出 `blueprint.json` 和 `image_handoff.json`
2. `run_copy_to_image.py` 读取 `image_handoff.json`
3. 顺序调用 `ip-image-skill`
4. 把图片结果和 manifest 落到本地

这不是完整 pipeline，只是先验证：

- 文案结构是否真的能喂给图片 skill
- 图片 skill 是否能按文案 handoff 批量执行

## 使用方式

先准备好：

- 一份 `image_handoff.json`
- 图片 API 环境变量

再执行：

```powershell
python scripts/run_copy_to_image.py --handoff E:\Plans for 2026\ip-skills\outputs\copy_skill_smoke\image_handoff.json --out E:\Plans for 2026\ip-skills\outputs\copy_to_image_smoke
```

## 输出

输出目录会包含：

- 多张图片结果
- `copy_to_image_manifest.json`

## 当前定位

这个 bridge 只做最小串联，不替代后续更完整的 pipeline。
