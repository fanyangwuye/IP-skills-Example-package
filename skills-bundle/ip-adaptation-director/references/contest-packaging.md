# 赛事包装模板

按需读取。用于光帆杯、AIGC赛事、IP改编比赛投稿说明和项目包装。

## 评分维度常用包装

- **IP适配性**：保留核心世界观、角色关系味道、视觉符号、情绪母题。
- **改编创新性**：新视角、新角色、新事件、新互动结构，而不是复刻原作。
- **AIGC技术应用**：剧本拆解、角色一致性、15秒分段、参考图锁定、视频生成、声音字幕、剪辑流程。
- **出海商业化价值**：低门槛理解、强视觉钩子、互动体验、社群传播、周边延展。

## 300字投稿说明模板

```text
本作品基于《IP名》的核心世界观与情绪气质，选择从【切入视角】进入故事，讲述【主角/玩家】在【核心事件】中面对【主要冲突】的过程。改编重点不是复刻原作名场面，而是提取其【核心资产/视觉符号/主题母题】，转化为更适合AIGC视频表达的短片结构。

制作上，作品采用【视频模型/流程】，将剧情拆分为【数量】个15秒节点，分别锁定角色、场景、道具、光影与动作连续性，保证生成画面的一致性与可剪辑性。同时，方案预留互动影游与周边衍生空间，可进一步拓展为【互动Demo/角色卡/道具周边/海外传播内容】。

整体目标是用AIGC方式验证该IP在短视频、互动体验与年轻用户传播中的新可能。
```
