# 周边衍生玩法库

按需读取。用于生成 IP 周边衍生包。

## 周边类型

1. **角色类**：角色卡、立牌、徽章、贴纸、明信片、海报、头像框。
2. **道具类**：核心道具复刻、笔记本、文件袋、钥匙扣、摆件、文具。
3. **组织/身份类**：阵营徽章、身份卡、通行证、成员证、任务卡。
4. **世界观类**：地图、档案袋、设定集、规则手册、怪谈/案件卡。
5. **互动类**：剧情选择卡、阵营测试卡、抽签牌、解谜包、桌游卡牌。
6. **数字周边**：动态壁纸、头像框、电子角色卡、可分享剧情卡。
7. **低成本宣发类**：海报、贴纸、明信片、社群表情包。
8. **高概念展示类**：概念盲盒、收藏册、限定礼盒、AR卡。


## 选择原则

- 先选“识别度强 + 成本低 + 可拍照传播”的品类。
- 原创 IP 可以更大胆；赛事 IP 默认写“概念周边/展示设想”。
- 周边必须来自 IP 资产，不要凭空做泛泛文创。
