# Combat Styles / 武打风格分型（运镜+招式逻辑）

Read this when the action scene has a specific combat style. Each style has its own move logic, camera grammar, and impact flavor. Pick one, stay consistent. All styles use the platform-safe impact rules from `fight-choreography.md` (no blood/gore).

> Rule: style drives BOTH the moves AND the camera. A wuxia duel and a CQB gunfight are filmed completely differently.

---

## 1. 武侠 / 吊威亚 (Wuxia / Wire-fu)

- **招式逻辑**：飘逸但可追踪；腾跃、踏枝、剑气、轻功落地有姿态。
- **运镜**：大量横移跟拍、升降镜头、剑锋特写接全景；落地给低机位。
- **打击感**：剑气破风、衣袖鼓动、落叶/水花/尘土做动量证据；交击出火花。
- **节奏**：写意慢起 → 凌厉快攻 → 定格亮相。
- **安全**：刀剑接触给火花/布料反应，不给见血。
```text
robe sleeves flowing with momentum, sword arc through rain, lands on wet stone with splash, balanced stance, sparks on blade contact, no blood, stylized wuxia
```

## 2. 现代格斗 / 拳脚 (Modern Hand-to-Hand)

- **招式逻辑**：步法、护手、重心转移、肩部发力、贴身缠斗。
- **运镜**：手持中景（可控不乱）、低机位重拳、命中切近景看受击形变。
- **打击感**：命中靠头部轻甩+踉跄+撞墙/桌，闷响卡点，镜头微震。
- **节奏**：试探→交火→压制→反击逆转。
- **安全**：不写真实搏击教学、不写要害打法；只写电影化动作。
```text
tight footwork, shoulder rotation, punch lands with head snap, opponent staggers into concrete wall, dust bursts, handheld medium shot, faces clear, non-graphic PG-13 action
```

## 3. 醉拳 / 谐趣武打 (Drunken / Comedic Kung-fu)

- **招式逻辑**：东倒西歪的破绽即杀招，节奏错拍，借力卸力。
- **运镜**：跟摇配合摇晃身姿，中景为主，节奏卡笑点不卡狠。
- **打击感**：滑稽闪避+借势反打，环境道具(板凳/酒坛)参与。
- **节奏**：装醉(松) → 突然精准(紧) → 又松回去。
- **安全**：喜剧基调天然安全，强调灵巧不强调伤害。
```text
loose swaying stance, stumble that turns into precise counter, uses bench and jar, playful dodges, rhythmic comedic timing, stylized
```

## 4. 枪战 / CQB (Gunfight / Close Quarters)

- **招式逻辑**：掩体、走位、交替推进、换弹节奏、视线与枪线。
- **运镜**：掩体后OTS、低机位推进、枪口火光做节拍、空间地理优先。
- **打击感**：枪口火光+弹着环境反馈(碎玻璃/火花/木屑)+角色找掩体的紧迫。
- **节奏**：对峙→交火→压制→换弹空档→突进。
- **安全**：**重点规避**——不写真实战术教学、不写枪械操作细节、不写中弹见血；用火光、跳弹环境反馈、走位紧张感表现，命中改为"被压制后退/扑向掩体"。
```text
takes cover behind concrete pillar, muzzle flash synced to beat, glass shatters, sparks off metal, tense advance between cover, no graphic injury, no real weapon handling detail, cinematic stylized
```

## 5. 刀剑对决 (Sword / Blade Duel)

- **招式逻辑**：刀位是连续性锚点；格挡-招架-还击的转化。
- **运镜**：接触点近景+火花，再拉回中景看姿态；刀锋轨迹清晰。
- **打击感**：金属交击火花、破风、刀身震动、布料割裂(非身体)。
- **节奏**：试剑→缠斗→锁刀→借力破势。
- **安全**：割到衣物/绳/物件，不割人见血。
```text
right-hand blade held low, parry from upper left, sparks at contact, blades lock at chest height, cloth sliced not flesh, no blood, stylized duel
```

## 6. 追逐战 (Chase — 非对抗动作)

- **招式逻辑**：地理优先，障碍升级，跑酷式翻越。
- **运镜**：背后跟拍、侧向横移、障碍处切换景别。
- **打击感**：擦碰、翻越、险些被抓的紧迫感(near-miss)。
- **节奏**：起跑→障碍→险境→暂时摆脱/被困。
- **安全**：天然安全，强调速度与空间。
```text
narrow alley wet ground, vaults fallen bicycle, camera follows behind, near-miss grab, ends at locked gate, kinetic stylized chase
```

---

## Cross-Style Rules

- **一场戏锁一个主风格**，混合需有理由（如武侠+现代=穿越/复古未来）。
- 风格决定运镜，别用枪战的手持去拍武侠的飘逸。
- 所有风格统一挂安全尾巴：`无血腥，无伤口特写，无真实战术/武器教学，风格化非写实，PG-13`。
- 打击感五要素（顿帧/形变/速度/音画/环境）对所有风格通用，只是"环境反馈"的素材换皮（水花 vs 火花 vs 尘土 vs 碎玻璃）。
