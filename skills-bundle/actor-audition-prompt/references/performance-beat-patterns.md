# Performance Beat Patterns / 表演节拍模板库

Read this when building the Acting Spine and 15-second segment performance beats. Pick the closest pattern, then localize to the actual script. Each pattern gives: objective, obstacle, subtext, leak behavior, emotional arc, body/breath/eye cue, common mistake.

> Rule: never write abstract emotion ("她很伤心"). Always convert to visible behavior the model can render.

---

## 1. 哭戏 (Cry Scene)

- **objective**: 想压住眼泪、维持体面，或想被看见、被理解。
- **obstacle**: 不能哭出来 / 必须哭出来的处境冲突。
- **subtext**: 表面在说别的，真正在崩。
- **leak behavior**: 喉结滚动、下巴收紧、鼻翼抽动、眼眶先红后湿、手攥衣角。
- **emotional arc**: 强撑 → 触发点 → 眼眶失守 → 一次吞咽压回 → 决堤或硬憋住。
- **body/breath/eye**: 呼吸先浅后断续，眨眼频率升高，视线躲闪后定住。
- **common mistake**: 一上来就嚎，没有"忍"的层次，模型只会糊一脸泪。

## 2. 爆发戏 (Explosion / Outburst)

- **objective**: 夺回控制权、逼对方让步。
- **obstacle**: 长期压抑、忌惮后果。
- **subtext**: 愤怒之下常是恐惧或受伤。
- **leak behavior**: 声音先抖后炸、青筋、指节发白、前倾逼近。
- **emotional arc**: 隐忍 → 最后一根稻草 → 临界停顿 → 爆 → 爆后空洞/后悔。
- **body/breath/eye**: 爆发前一个憋气，瞳孔放大，爆后肩膀塌。
- **common mistake**: 全程高音量，没有"临界停顿"，缺乏对比就不可信。

## 3. 克制戏 (Restraint / Suppression)

- **objective**: 不暴露真实情绪、保护自己或对方。
- **obstacle**: 情绪强度高，几乎压不住。
- **subtext**: 越平静越危险。
- **leak behavior**: 极小动作——一次深呼吸、指尖轻颤、嘴角下压、眼神短暂失焦。
- **emotional arc**: 平静 → 内部翻涌 → 一个微泄 → 重新压回。
- **body/breath/eye**: 刻意放慢的呼吸，长时间不眨眼后猛眨一次。
- **common mistake**: 演成面瘫；克制不是没表情，是"压住的表情"。

## 4. 反派试戏 (Villain)

- **objective**: 支配、玩弄、试探猎物。
- **obstacle**: 需要伪装、不能过早暴露。
- **subtext**: 礼貌之下是威胁。
- **leak behavior**: 慢条斯理、微笑不到眼睛、停顿用来施压、突然的安静。
- **emotional arc**: 松弛掌控 → 收紧 → 露獠牙 → 收回伪装。
- **body/breath/eye**: 呼吸稳定到反常，视线扫描对方弱点，头部微倾。
- **common mistake**: 一直龇牙咧嘴瞪眼，廉价；高级反派靠"松弛+突变"。

## 5. 古装试戏 (Period / Costume Drama)

- **objective**: 身份、礼制、权谋下的隐忍或锋芒。
- **obstacle**: 不能逾矩，话不能说透。
- **subtext**: 台词三分，潜台词七分。
- **leak behavior**: 收敛的体态、袖手、眼波、欲言又止、停顿后改口。
- **emotional arc**: 端着 → 暗涌 → 借物/借礼仪宣泄 → 复位。
- **body/breath/eye**: 身形挺直，动作幅度小而准，眼神戏重。
- **common mistake**: 用现代生活化小动作，破古装气质；台词太白。

## 6. 现代短剧试戏 (Modern Short Drama)

- **objective**: 强情绪钩子、3秒抓人。
- **obstacle**: 时长短，必须快速立人设。
- **subtext**: 反转前埋一个"看似正常"。
- **leak behavior**: 一个突兀停顿、一个表情切换、一句重音台词。
- **emotional arc**: 日常 → 冲突点 → 情绪切换 → 钩子结尾。
- **body/breath/eye**: 节奏快、表情对比强、机位贴脸。
- **common mistake**: 铺垫太长，短剧没耐心；情绪不够直给。

## 7. 双人对手戏 (Two-Hander)

- **objective**: 双方各有目标，互相博弈。
- **obstacle**: 对方不配合 / 隐藏意图。
- **subtext**: 听对方话的反应戏比说台词更重要。
- **leak behavior**: 反应镜头——听到关键信息时的微表情变化。
- **emotional arc**: 试探 → 攻防 → 力量倒转 → 新平衡。
- **body/breath/eye**: 视线交锋、身体距离变化、谁先移开。
- **common mistake**: 只演自己说话的部分，不演"听"；反应戏缺失。

## 8. 审讯戏 (Interrogation)

- **objective**: 一方挖真相，一方守底线。
- **obstacle**: 压力、疲惫、心理战。
- **subtext**: 说谎者的破绽 / 审讯者的诱导。
- **leak behavior**: 吞咽、视线下移、停顿过长、不自觉重复、手部小动作。
- **emotional arc**: 镇定 → 被戳中 → 破绽 → 重新伪装或崩。
- **body/breath/eye**: 呼吸节奏变化是关键，灯光压迫。
- **common mistake**: 破绽太明显，没有"伪装-泄露-再伪装"的反复。

## 9. 告别戏 (Farewell)

- **objective**: 想好好告别 / 想留住 / 想体面离开。
- **obstacle**: 时间、处境不允许、说不出口。
- **subtext**: 说的是"再见"，想的是"别走"。
- **leak behavior**: 转身又回头、欲拥抱又收手、声音突然轻、笑里带哭。
- **emotional arc**: 强装轻松 → 触动 → 失守一瞬 → 强行收住转身。
- **body/breath/eye**: 长久对视、一次深吸气、最后一眼。
- **common mistake**: 一味煽情大哭；克制的告别更催泪。

## 10. 崩溃戏 (Breakdown / Collapse)

- **objective**: 已无力维持，彻底失控。
- **obstacle**: 此前一直在硬撑。
- **subtext**: 崩溃前往往有一个荒谬的平静。
- **leak behavior**: 笑着哭、瘫坐、抓头、语无伦次、身体失去支撑。
- **emotional arc**: 撑 → 最后防线被破 → 失重 → 全面崩 → 力竭后的空。
- **body/breath/eye**: 呼吸过度换气、肢体失控、眼神涣散。
- **common mistake**: 没有"撑"的铺垫，崩溃就没有重量。
