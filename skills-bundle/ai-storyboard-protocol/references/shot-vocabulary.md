# Shot Vocabulary / 统一镜头术语表

Read this so all three skills use the SAME words for shot size, angle, movement, and eyeline. Consistent vocabulary = cards stitch without translation.

---

## 景别 (Shot Size)

| 缩写 | 中文 | 用途 |
|---|---|---|
| ELS | 大远景 | 确立环境/空间总览（establishing） |
| LS | 远景/全景 | 全身+环境，交代走位 |
| MLS | 中远景 | 膝盖以上，群戏站位 |
| MS | 中景 | 腰以上，对话主力 |
| MCU | 中近景 | 胸以上，情绪升温 |
| CU | 近景 | 头肩，情绪/反应 |
| ECU | 特写 | 眼/嘴/手/道具，强情绪或细节 |
| OTS | 过肩 | 对话双人关系，锁屏幕方向 |
| POV | 主观 | 角色视角 |
| insert | 插入镜头 | 道具/动作细节，也用于过渡越轴 |

## 机位角度 (Angle)

- eye level 平视：中性、对等。
- high angle 俯视：弱化主体、施压。
- low angle 仰视：强化主体、威胁感。
- dutch angle 斜角：失衡、紧张、混乱。
- bird's eye 顶视：空间总览/宿命感。

## 镜头运动 (Movement)

- static 固定：脆弱情绪、对峙、稳。
- dolly in / push 推：揭示、压迫、聚焦。
- dolly out / pull 拉：抽离、孤独、揭示环境。
- track / follow 跟拍：跟随移动主体。
- pan 摇：横向扫视、连接两点。
- tilt 俯仰摇：上下连接。
- handheld 手持：纪实感、紧张、动作戏。
- whip pan 甩镜：快速转场/动作衔接。
- crane 升降：情绪起落、规模感。
- arc 环绕：强调关系/对峙张力。

## 视线 (Eyeline)

- 朝向：朝左 / 朝右 / 朝镜头 / 背对。
- 高度：平视 / 俯视(朝下) / 仰视(朝上)。
- 对视：两人视线互补相对 = 对视成立。
- 离线视线：故意不对视，表疏离/隐瞒。

## 节奏 (Rhythm by Duration)

- 1 min（4×15s）：少切、单一情绪焦点，CU/MCU 为主。
- 2-3 min：MS/CU/insert 受控交替。
- 4-5 min：加走位、前后景、POV、监视器视角。
- 10 min：按段落分配，避免 40 节点同一景别。

## 使用约定

- 每 15s 节点最多一个主运镜。
- 脆弱情绪用 static，揭示/压迫用慢推。
- insert 只给有意义的道具/动作，也可当越轴过渡。
- 对话戏 OTS + 屏幕方向锁定是防越轴主力。
