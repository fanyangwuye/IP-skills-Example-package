# Storyboard Card Template / 标准分镜卡模板

Read this for the copyable blank card and two worked examples (one dialogue, one action). All three skills emit THIS structure so cards stitch together.

---

## Blank Card (copy this)

```markdown
### 镜头 N [MM:SS - MM:SS]
- **剧情作用**：
- **在场角色**：A=[标签/锁ID]，B=[...]，C=[...]
- **轴线**：[A↔B 轴线 / 移动轴线]，机位停在[轴线哪一侧]
- **屏幕方向**：A 屏幕左·朝右；B 屏幕右·朝左
- **画面**：景别[WS/LS/MS/CU/ECU] / 机位[平/俯/仰/OTS/dutch] / 运动[static/dolly/track/handheld/whip] / 主体位置
- **视线匹配**：[谁看谁]，视线[朝向+高度]，是否对视
- **表演/动作**：起始姿态 → 动作 → 结束姿态
- **连续性**：上镜结束态[...] → 本镜起始态[...] → 本镜结束态[...] → 下镜交接[...]
- **参考图绑定**：锁脸[...]/锁衣[...]/锁景[...]/锁道具[...]/锁风格[...] + 禁用[...]
- **声音字幕**：对白[...] / 旁白[...] / 环境声[...] / 字幕[需要/不需要+节奏]
- **【Seedance字段】**：
  - 中文提示词：
  - English prompt：
  - 负面约束：
  - 失败重抽建议：
```

## Example A — 对话戏 (Dialogue, filled by actor-audition-prompt)

```markdown
### 镜头 3 [00:30 - 00:45]
- **剧情作用**：女主识破男主在说谎，关系第一次裂开。
- **在场角色**：A=女主(face_lock_01)，B=男主(face_lock_02)
- **轴线**：A↔B 横向轴线，机位停在轴线上方一侧，全场不跳。
- **屏幕方向**：A 屏幕左·朝右；B 屏幕右·朝左。
- **画面**：MS 双人 → 切 A 的 OTS 近景；平视；static + 极缓推。
- **视线匹配**：A 看 B（朝右、平视）；切 B 时 B 看 A（朝左、平视），互补对视。
- **表演/动作**：A 起始端坐微笑 → 听到破绽后笑意僵住、指尖收紧 → 结束时视线下移一次再抬起逼视。
- **连续性**：上镜结束态[两人对坐，A 持茶杯] → 本镜起始态[A 茶杯仍在手、B 身体微后靠] → 本镜结束态[A 放下茶杯、前倾] → 下镜交接[B 接话，A 维持前倾]。
- **参考图绑定**：锁脸 A/B 各自；锁衣不变；锁景同一茶室；禁用新增人物/道具。
- **声音字幕**：对白 A"你刚才说你没去过那里"[重音"没去过"][停顿0.5s]；环境声 安静室内+轻茶杯声；字幕仅关键句。
- **【Seedance字段】**：
  - 中文提示词：茶室内对坐两人，女主屏幕左面朝右、男主屏幕右面朝左，女主从端坐微笑到笑意僵住、指尖收紧、视线下移再抬起逼视，OTS近景平视极缓推，安静室内…
  - English prompt：two people seated in a tea room, woman screen-left facing right…
  - 负面约束：无越轴、无屏幕方向翻转、无新增人物、无表情夸张
  - 失败重抽建议：若视线错位→强调"A朝右看B、B朝左看A互补对视"
```

## Example B — 动作戏 (Action, filled by seedance-action-director)

```markdown
### 镜头 5 [01:00 - 01:15]
- **剧情作用**：追击中男主被逼到死巷，关系反转前的最后压迫。
- **在场角色**：A=男主(face_lock_02)，B=追兵(generic_lock_03)
- **轴线**：移动轴线=向右，机位守右移方向一侧。
- **屏幕方向**：A 屏幕左向右逃；B 屏幕左后方追（同向右运动）。
- **画面**：LS 跟拍 → 死巷处切 MS；低机位；handheld track。
- **视线匹配**：A 回头瞥 B（朝左后），B 紧盯 A（朝右前）。
- **表演/动作**：A 起始全力右奔 → 撞见死巷急刹、抓铁门发现锁住 → 结束背贴铁门转身面对 B。
- **连续性**：上镜结束态[A 越过自行车] → 本镜起始态[A 仍向右冲、追兵五米后] → 本镜结束态[A 背贴锁住的铁门、B 三米外] → 下镜交接[B 扑近，A 背门]。
- **参考图绑定**：锁脸 A；锁衣 湿黑外套破左袖；锁景 窄巷；锁道具 铁门锁；禁用 武器凭空出现。
- **声音字幕**：无台词；环境声 急促脚步+雨；A 一次喘息；无字幕。
- **【Seedance字段】**：
  - 中文提示词：窄巷雨夜，男主湿黑外套破左袖屏幕左向右逃，撞见死巷急刹抓锁住的铁门，背贴铁门转身面对三米外追兵，低机位手持跟拍…
  - English prompt：narrow rainy alley, man in wet black coat torn left sleeve running rightward…
  - 负面约束：无瞬移、无武器凭空出现、无服装变色、无越轴
  - 失败重抽建议：若方向乱→重申"全程向右运动，追兵在左后方"
```
