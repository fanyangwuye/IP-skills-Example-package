# Continuity Rules / 跨镜头连续性规则（对白+动作通用）

Read this when building any multi-node sequence. These rules apply to BOTH dialogue (actor-audition-prompt) and action (seedance-action-director) cards, so they stitch into one continuous video.

> Core chain: 上镜结束态 → 本镜起始态 → 动作/表演转化 → 本镜结束态 → 下镜起始指令。链断 = 像不相关片段硬拼。

---

## Continuity Anchors (每镜必追踪)

1. **身体姿态**：站/蹲/跪/坐/倒/靠。
2. **朝向**：朝镜头/朝对手/背对/朝左右。
3. **屏幕方向**：A 在屏幕左还是右（全场锁定，防越轴）。
4. **空间关系**：谁离谁多远、谁高谁低、靠近哪个物体。
5. **视线落点**：看谁/看哪、视线高度。
6. **道具/武器状态**：杯在手、剑低垂、门锁住、枪入鞘。
7. **服装/损伤状态**：湿衣、破袖、沾尘、血痕（风格化）。
8. **环境状态**：碎桌、开门、雨势、散落纸张、烟位。
9. **情绪/动作节奏**：对峙/退守/反扑/力竭。

---

## Handoff Rule

每张卡的"本镜起始态"必须**逐字复制**上一张卡的"本镜结束态"：

```text
镜头3结束态：A 放下茶杯、前倾，B 微后靠。
镜头4起始态：必须从"A 前倾、茶杯已放下，B 微后靠"开始。
```

不许在交接处偷偷改姿态、改距离、改屏幕方向。

## Common Bugs（对白+动作通用）

### 1. 站位瞬移 / 距离重置
- Bad：上镜两人对坐，下镜突然站在门口。
- Fix：写出移动过程，或保持原位。

### 2. 屏幕方向翻转（越轴）
- Bad：A 上镜在左，下镜跑到右，无过渡。
- Fix：锁定屏幕方向；要换侧用合法过渡（见 multichar-axis-rules.md）。

### 3. 视线断裂
- Bad：对话双方同时朝右看。
- Fix：视线互补，A 朝右看 B、B 朝左看 A。

### 4. 道具/武器瞬现瞬移
- Bad：上镜空手，下镜挥枪。
- Fix：上镜交代枪靠墙，下镜伸手抓起。

### 5. 环境重置
- Bad：上镜桌子打碎，下镜完好。
- Fix：保留碎板、木屑、水渍、烟位。

### 6. 服装漂移
- Bad：湿黑外套变白袍。
- Fix：每镜提示词重复 same costume, no change。

### 7. 情绪/动作断层
- Bad：上镜暴怒，下镜无诱因突然平静。
- Fix：补诱因，用可见行为过渡，不写抽象情绪。

### 8. 一镜塞太满
- Bad：15s 里又跑又打又说三段台词。
- Fix：拆节点，每镜一个主动作/主情绪。

## Pre-Output Checklist

- [ ] 每镜起始态 == 上镜结束态？
- [ ] 屏幕方向全场一致？换侧有过渡？
- [ ] 视线互补、不同向？
- [ ] 道具/武器位置一致？
- [ ] 距离/方向可追踪？
- [ ] 环境损伤/水/尘/烟状态保留？
- [ ] 新道具/新角色是否有意引入并交代方向？
- [ ] 每镜只有一个主动作意图？
- [ ] 每镜结束都给了可用的下镜起始姿态？

任一 high 级（瞬移/越轴/视线同向/武器瞬现）→ 打回。
